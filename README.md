# Spam Detection System

A machine learning-based spam detection system using:
- Bidirectional LSTM (deep learning)
- TF-IDF + Logistic Regression
- FastAPI backend
- Streamlit frontend

## Features
- Detect spam vs ham
- Confidence scoring
- Risk levels (Safe / Suspicious / Dangerous)
- Explainability (top signals + reason)
- Logging system

## Models Used
- LSTM (context understanding)
- Logistic Regression (keyword baseline)
- Ensemble decision system

## Tech Stack
- Python
- FastAPI
- Streamlit
- TensorFlow / Keras
- Scikit-learn
